
# Smart Study Assistant 📚

A comprehensive AI-powered platform designed to streamline the learning process, organize study materials, and provide intelligent assistance for students.

## 🚀 Features
* **AI Chat:** Ask questions and get instant, context-aware explanations using the Groq API.
* **PDF Analyzer:** Upload study materials and interact with your documents.
* **Study Planner:** Organize your learning schedule efficiently.
* **Quiz Generator:** Create practice tests to reinforce your knowledge.
* **Notes Management:** Keep all your study notes in one place.

## 🛠 Tech Stack
* **Backend:** Flask (Python)
* **AI Integration:** Groq API
* **Frontend:** HTML5, CSS3, Bootstrap

## 📦 Installation
1. Clone the repository: `git clone https://github.com/Suryakumar00/smart-study-assistant.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `GROQ_API_KEY` to your environment variables.
4. Run: `python app.py`
